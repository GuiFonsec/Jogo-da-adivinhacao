Bom dia/ tarde/ noite!
Este é o Jogo de adivinhação desenvolvido completamente na linguagem C.

A objetivo do jogo é que o jogador tente descobrir um número secreto selecionado aleatoriamente pelo computador, tendo 2 modos, um
com infinitas tentativas e outro com dificuldades que o jogador poderá escolher.

Para rodar o jogo, é necessário abrir o prompt de comando do seu computador e procurar o diretório onde o jogo está armazenado ("cd" no caso do Windows, Exemplo: cd C:\Users\Administrador\Jogo da adivinhacao), e executar
o arquivo .exe escrevendo-o e apertando a tecla enter.

adivinhacaoinfinito.exe para o modo infinito.
adivinhacaodificuldades.exe para o modo com dificuldades.

Tenha um bom jogo!
